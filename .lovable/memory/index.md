# Project Memory

## Core
Financial Nexus app - cyberpunk/neon aesthetic. Primary blue, secondary green, destructive red.
Dark theme default. English/Arabic i18n with RTL support.
Lovable Cloud backend with Supabase: profiles, transactions, investments, savings_goals, budgets, lendings tables.
Auth: Email + Google + Apple. All dashboard routes are protected.
Migrated from Manus (GitHub: suhaibnuwaila61-bit/financial_advisor).
